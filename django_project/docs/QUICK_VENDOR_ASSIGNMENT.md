# Quick Vendor Assignment Reference

## 🎯 Quick Steps to Assign a Vendor

### For Buyers & Admins

**Step 1: Open the PR**
- Go to Dashboard
- Click on any PR

**Step 2: Click "Update PR"**
- Button visible on PR detail page

**Step 3: Select Vendor**
- Find "Assign Vendor" dropdown
- Choose from approved vendors
- Only active vendors shown

**Step 4: Save**
- Click "Save Changes"
- Done! ✓

---

## 📋 Vendor Assignment Checklist

Before assigning a vendor, check:

- [ ] Vendor is approved and active
- [ ] Vendor rating is acceptable (check stars)
- [ ] Vendor serves the PR category
- [ ] Payment terms are suitable
- [ ] Vendor has capacity for new orders

---

## 🔍 Where to Find Vendors

### View All Vendors
```
Navigation → Vendors → Browse List
```

### Search Vendors
```
Vendors → Search Box → Enter Name/Email
```

### Filter by Status
```
Vendors → Status Dropdown → Select Active
```

---

## 👥 Who Can Do What?

| Role | Assign Vendor | Edit Vendor | View Vendor |
|------|--------------|-------------|-------------|
| **Admin** | ✅ Yes | ✅ Yes | ✅ Yes |
| **Buyer** | ✅ Yes | ❌ No | ✅ Yes |
| **Requester** | ❌ No | ❌ No | ❌ No |
| **Vendor** | ❌ No | ❌ No | ✅ Own Only |

## 🚀 Quick Actions

### Assign Vendor from PR Detail
```
PR Detail → Update PR → Assign Vendor → Save
```

### View Vendor's Current Orders
```
Vendors → Click Vendor → Orders Tab
```

### Check Vendor Rating
```
Vendors → View List → Rating Column (⭐⭐⭐⭐⭐)
```

### Reassign to Different Vendor
```
PR Detail → Update PR → Change Vendor → Save
```

---

## ⚠️ Common Issues

### "No vendors in dropdown"
**Solution**: Check if any vendors are approved and active

### "Cannot save vendor assignment"
**Solution**: Ensure you're a buyer or admin

### "Vendor not showing"
**Solution**: Vendor must be approved by admin first

## 💡 Pro Tips

**Tip 1**: Check vendor rating before assigning
- Higher stars = better performance

**Tip 2**: Review vendor's past orders
- Go to Vendor Detail → Orders Tab

**Tip 3**: Add notes when assigning
- Document why you chose this vendor

**Tip 4**: Update PR status after assignment
- Change from "Open" to "Pending"

**Tip 5**: Monitor vendor performance
- Track delivery times and payment status
